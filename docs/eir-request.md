# EIR request: Disabled / Blue Badge parking bay locations in Newham

**Authority:** London Borough of Newham  
**Suggested channel:** WhatDoTheyKnow or Newham Council FOI/EIR contact form  
**Subject:** Environmental Information Regulations request — disabled parking bay locations and source files

Dear London Borough of Newham,

Please treat this as a request under the Environmental Information Regulations 2004.

I am requesting the most complete dataset you hold for disabled / Blue Badge parking bays within the London Borough of Newham.

Please provide the information in any format you hold, including but not limited to:

1. GIS files, shapefiles, GeoJSON, CSV, Excel, database exports, CAD files, PDF plans, schedule plans, traffic order schedules, map layers, contractor-held files, or any other electronic format.
2. The location of each disabled / Blue Badge bay, including street name and coordinates where available.
3. Whether each bay is dedicated, advisory, resident-specific, enforceable, time-limited, or subject to any other condition.
4. Any source metadata, including last updated date, order reference, map tile reference, or contractor system name.
5. Any current or historical files used by council officers, parking contractors, highways teams, enforcement teams, GIS teams, or external suppliers to locate or manage disabled parking bays.

If the information is held by a contractor, map supplier, highways partner, enforcement contractor, or GIS platform provider on behalf of the council, please treat that as information held by the authority for the purposes of the EIR and request it from the relevant contractor.

I prefer the data in machine-readable format such as GeoJSON, shapefile, CSV, Excel, or database export. If no machine-readable file exists, please provide the most detailed plans or schedule documents available.

If any part of this request is refused, please provide the legal basis for refusal and release the remaining information that can be disclosed.

Yours faithfully,

[Your name]
